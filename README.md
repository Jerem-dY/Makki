# Le lexique Libanais de Makki

![logo](/html/images/logo.png)

Le lexique de Makki, une compilation lexicale de grande ampleur, constitue un précieux témoignage du parler libanais avec un accent particulier sur la région du grand Sud. Cette dernière, étendue sur plus de 3000 pages dactylographiées, résulte du travail minutieux de l'ingénieur libanais Hassan Makki. Originaire d'Erkay, dans le sud-est de la ville de Saïda, Makki a consacré à partir de la fin des années 1990 près d'une quinzaine d'années à cette collecte linguistique.
